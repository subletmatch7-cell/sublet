# SubletMatch

SubletMatch is a modern rental subletting platform that connects listers with verified renters in real time. It provides a seamless experience for listing, discovering, and booking short-term rental properties with built-in analytics, authentication, and admin management tools.

Built with scalability, security, and performance in mind.

---

## Overview

SubletMatch helps:

- Listers publish and manage rental listings
- Users discover and inquire about available properties
- Admins monitor platform activity and demand metrics

The system is designed with clean architecture and production-ready practices.
